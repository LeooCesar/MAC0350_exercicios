# Music Ranker - WebMAC 🎵

A Music-Ranker web app built to integrate HTML, CSS, JavaScript, FastAPI, SQL, and HTMX. 

## Features

In this app, users can:
* **Add** their favorite songs along with their grades and albums (Create).
* **View** all their songs sorted by grade with infinite scroll (Read).
* **Update** any song's grade (Update).
* **Delete** tracks from their list (Delete).

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript, HTMX
* **Backend:** Python, FastAPI
* **Database:** SQL
